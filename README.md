# Sport Rental Database

Проект базы данных пункта проката спортивного инвентаря.

## Выполненные задания

1. Разработка ER-схемы базы данных.
2. Создание таблиц.
3. Заполнение базы тестовыми данными.
4. Проведение ABC-анализа спортивного инвентаря.

## Таблицы

- Clients
- Inventory
- Employees
- Booking
- Returns

## Анализ

Проведен ABC-анализ доходности спортивного инвентаря на основе данных об аренде.
Результаты анализа:
Категория A:
- Велосипед (4000 руб.)

Категория B:
- Палатка (2500 руб.)

Категория C:
- Самокат (2100 руб.)

Наибольший доход приносит велосипед, поэтому данный инвентарь относится к категории A и требует приоритетного контроля. Палатка относится к категории B. Самокат относится к категории C и приносит наименьший доход.
